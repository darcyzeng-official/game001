# 写给殷悦的游戏:一起玩吧

一个**你永远无法逃脱的剪刀石头布网页游戏**。  

⚠️ **注意事项**  
- 必须点击页面按钮才能播放背景音乐（BGM）。  
- 输三次后，屏幕将被乱码覆盖，最终宣布你无法逃脱。  
- 请在现代浏览器中打开，建议 Chrome 或 Edge。  

## 🎮 游戏玩法
1. 打开 [游戏网页](https://darcyzeng-official.github.io/game001/)  
2. 页面会问你：“殷悦……和我一起玩个游戏吗？”  
   - 选择“不要”会被步步紧逼，最终只剩“要玩”。  
   - 选择“要玩”立即进入游戏。  
3. 游戏规则：剪刀、石头、布  
4. 输三次后，屏幕会变成满屏乱码，宣布你逃不掉
